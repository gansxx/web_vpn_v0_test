# 多平台客户端指南 📱

Z加速支持所有主流设备和操作系统。本指南将为您提供各平台的详细配置教程。

---

## 🖥️ Windows 平台

### 推荐客户端: v2rayN
**系统要求:** Windows 7/8/10/11

#### 📥 下载与安装
1. **下载地址**: [v2rayN 官方版本](https://wwbi.lanzoue.com/iPwsU36smqbc)
2. **安装步骤**:
   - 解压下载的zip文件
   - 双击 `v2rayN.exe` 启动
   - 如出现安全提示，选择"仍要运行"

#### ⚙️ 配置步骤
1. **导入订阅**:
   - 点击"配置文件" → "从剪贴板导入分享链接"
   - 或者：右键系统托盘图标 → "从剪贴板导入批量URL"

2. **系统代理设置**:
   - 点击"系统代理" → "自动配置系统代理"
   - 选择"路由" → "绕过大陆路由"

3. **服务器选择**:
   - 双击延迟最低的服务器
   - 或右键选择"设为活动配置"

## 🍎 macOS 平台

### 推荐客户端: ClashX Pro
**系统要求:** macOS 10.12+

#### 📥 下载与安装
1. **获取客户端**: 联系客服获取macOS专用下载链接
2. **安装说明**:
   - 下载 `.dmg` 文件
   - 拖拽到Applications文件夹
   - 首次运行需要在"安全性与隐私"中允许

#### ⚙️ 配置步骤
1. **导入配置**:
   - 点击菜单栏图标
   - 选择"配置" → "从URL下载配置"
   - 粘贴订阅链接

2. **代理模式**:
   - 选择"规则判断"模式
   - 启用"设为系统代理"

3. **节点选择**:
   - 点击"代理" → "延迟测试"
   - 选择延迟最优节点

## 📱 iOS 平台

### 推荐客户端: Shadowrocket
**系统要求:** iOS 13.0+

#### 📥 下载与安装
1. **App Store购买**: 需要购买Shadowrocket应用（$2.99）
2. **备选方案**: 联系客服获取其他iOS客户端

#### ⚙️ 配置步骤
1. **添加订阅**:
   - 打开Shadowrocket
   - 点击右上角"+"
   - 选择"类型" → "Subscribe"
   - 粘贴订阅链接并保存

2. **连接设置**:
   - 选择服务器节点
   - 开启连接开关
   - 首次使用需要允许VPN配置

3. **规则配置**:
   - 进入"配置"页面
   - 选择"默认配置"
   - 启用"自动代理"

## 🤖 Android 平台

### 推荐客户端: v2rayNG
**系统要求:** Android 5.0+

#### 📥 下载与安装
1. **获取APK**: 联系客服获取Android客户端下载链接
2. **安装方法**:
   - 开启"未知来源"安装权限
   - 下载并安装APK文件

#### ⚙️ 配置步骤
1. **导入订阅**:
   - 点击右上角"+"
   - 选择"从剪贴板导入"
   - 或手动添加订阅URL

2. **连接配置**:
   - 选择服务器节点
   - 点击右下角连接按钮
   - 首次连接需要允许VPN权限

3. **路由设置**:
   - 进入"设置" → "路由设置"
   - 选择"绕过局域网及大陆地址"

## 🐧 Linux 平台

### 推荐客户端: v2ray-core + 图形界面
**系统要求:** Ubuntu 18.04+ / CentOS 7+

#### 📥 安装命令
```bash
# Ubuntu/Debian
sudo apt update
sudo apt install v2ray

# CentOS/RHEL
sudo yum install v2ray
```

#### ⚙️ 配置步骤
1. **配置文件**:
   ```bash
   sudo nano /etc/v2ray/config.json
   ```

2. **导入订阅配置**:
   - 联系客服获取Linux配置文件
   - 或使用第三方订阅转换工具

3. **启动服务**:
   ```bash
   sudo systemctl start v2ray
   sudo systemctl enable v2ray
   ```

## 📟 路由器平台

### 支持固件: OpenWrt / 梅林固件
**适用设备:** 支持第三方固件的路由器

#### 🔧 配置方法
1. **固件要求**:
   - OpenWrt 19.07+
   - 华硕梅林固件
   - 或其他支持插件的固件

2. **插件安装**:
   - 安装passwall或ssr-plus插件
   - 配置订阅链接
   - 设置透明代理

3. **网络配置**:
   - 全局代理或分流规则
   - DNS劫持防污染
   - 游戏加速模式

**注意**: 路由器配置较为复杂，建议有一定技术基础的用户尝试

## 🎮 游戏主机

### PlayStation / Xbox / Nintendo Switch

#### 🌐 通过路由器代理
1. **配置路由器VPN** (推荐)
   - 在路由器上配置VPN连接
   - 游戏主机连接该路由器WiFi
   - 自动获得加速效果

2. **手机热点分享**
   - 手机连接VPN
   - 开启热点分享
   - 游戏主机连接手机热点

#### 🔧 手动DNS设置
部分游戏主机支持修改DNS设置:
```
主DNS: 8.8.8.8
备用DNS: 1.1.1.1
```

## 🛠️ 平台特殊说明

### Windows 特别注意
- **防火墙**: 添加客户端到白名单
- **杀毒软件**: 可能误报，需要添加信任
- **管理员权限**: 部分功能需要管理员运行

### macOS 特别注意
- **网络权限**: 首次运行需要授权网络访问
- **系统代理**: 可能需要输入系统密码确认
- **Gatekeeper**: 需要在安全设置中允许运行

### iOS 特别注意
- **VPN配置**: 会在系统设置中创建VPN配置文件
- **企业证书**: 部分应用需要信任企业开发者
- **后台刷新**: 建议开启以保持连接稳定

### Android 特别注意
- **电池优化**: 建议关闭VPN应用的电池优化
- **自启动**: 开启自启动权限以自动连接
- **通知权限**: 允许显示连接状态通知
