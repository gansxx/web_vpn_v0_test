# VPN速度优化指南 ⚡
网速太慢？视频卡顿？本指南将帮助您最大化VPN连接速度，获得更佳的使用体验。

## 🚀 快速优化清单
立即尝试以下优化措施：
- [ ] 选择延迟最低的服务器节点
- [ ] 启用"绕过大陆路由"模式
- [ ] 关闭其他占用带宽的应用
- [ ] 使用有线网络替代WiFi

## 📊 网速测试与分析

### 测试网络速度
优化前请先测试当前网速：
1. **无VPN时**: 访问 [Speedtest](https://www.speedtest.net) 测试原始网速
2. **使用VPN时**: 连接VPN后再次测试对比
3. **记录数据**: 记录下载/上传速度和延迟数值

### 性能期望值
- **良好**: VPN速度为原始网速的70-80%
- **可接受**: VPN速度为原始网速的50-70%
- **需优化**: VPN速度低于原始网速的50%

## 🎯 服务器选择优化
### 选择最优节点
服务器选择是影响速度的关键因素：

#### 1. 延迟优先策略
```
优选顺序:
1. 延迟 < 50ms  - 极佳
2. 延迟 50-100ms - 良好
3. 延迟 100-200ms - 可接受
4. 延迟 > 200ms - 需更换
```

#### 2. 地理位置考虑
- **就近原则**: 选择地理位置最近的服务器
- **避开拥堵**: 避开使用高峰期的热门节点
- **专线服务器**: 优先选择标注"专线"的节点

#### 3. 实时监控
定期检查服务器状态：
- 使用v2rayN的服务器测速功能
- 关注服务器负载指示器
- 根据时段更换不同节点

## 🌐 网络环境优化

### 本地网络调优

#### 1. 连接方式优化
- **有线连接**: 使用网线连接，避免WiFi信号干扰
- **WiFi优化**: 使用5GHz频段，避开2.4GHz拥堵
- **信号强度**: 确保WiFi信号强度在-50dBm以上

#### 2. 路由器设置
```
路由器优化建议:
- 启用QoS带宽管理
- 设置VPN设备为高优先级
- 更新路由器固件到最新版本
- 重启路由器清除缓存
```

#### 3. 系统级优化
Windows系统优化：
```
网络适配器设置:
1. 设备管理器 → 网络适配器
2. 右键网卡 → 属性 → 高级
3. 关闭"节能模式"
4. 设置"接收缓冲区"为最大值
```

## 📈 应用层面优化

### 浏览器优化
- **多线程下载**: 启用浏览器并行下载
- **清除缓存**: 定期清理浏览器缓存
- **关闭插件**: 禁用不必要的浏览器扩展

### 流媒体优化
- **降低画质**: 将视频质量设置为720p或更低
- **预加载**: 启用视频预缓冲功能
- **专用节点**: 使用专为流媒体优化的服务器

### 下载优化
- **分时下载**: 避开网络高峰期进行大文件下载
- **多线程**: 使用支持多线程的下载工具
- **断点续传**: 启用断点续传功能

### 网络使用时段规划

最佳时段 (速度最快):

- **晚间娱乐**: 选择美国西海岸节点
- **深夜时段**: 选择欧洲节点
