# <font color="#1f497d">iOS用户重要提示</font>

> ⚠️ **重要**: iOS应用**不支持大陆Apple ID**下载，需要使用**美区Apple ID**才能在App Store中下载安装

## 📱 问题说明

由于App Store政策限制，hiddify的iOS客户端仅在**非国区App Store**上架，使用**中国大陆Apple ID**无法搜索和下载该应用。

## ✅ 解决方案

您需要切换到**美区Apple ID**来下载应用：

### 方法一：切换App Store账号（推荐）

1. **退出当前账号**
   - 打开App Store应用
   - 点击右上角头像
   - 滑动到底部，点击"退出登录"

2. **登录美区账号**
   - 使用您的美区Apple ID登录App Store
   - 搜索并下载"Z加速"应用

3. **使用完成后**
   - 可以退出美区账号
   - 重新登录您的中国区账号
   - **已下载的应用不会受影响**

### 方法二：创建美区Apple ID

如果您还没有美区Apple ID：

1. 访问 [appleid.apple.com](https://appleid.apple.com)
2. 创建新的Apple ID，**国家/地区选择"美国"**
3. 使用该账号登录App Store下载应用

## 💡 注意事项

- ✓ **仅需在App Store切换账号**，不影响iCloud等其他服务
- ✓ 应用下载后，可以切换回原账号，**应用仍可正常使用和更新**
- ✓ 建议保留美区账号用于后续应用更新

## ❓ 常见问题

**Q: 切换App Store账号会影响我的iCloud数据吗？**
A: 不会。App Store账号和iCloud账号是独立的，切换App Store账号不会影响您的照片、通讯录等iCloud数据。

**Q: 切换回中国区账号后，应用还能用吗？**
A: 可以。已下载的应用不受账号切换影响，可以正常使用。但应用更新时可能需要再次切换到美区账号。

**Q: 我没有美区Apple ID，创建需要美国信用卡吗？**
A: 不需要。创建美区Apple ID不需要绑定付款方式，可以选择"无"作为付款方式。
